## Notebook
This is simple notebook module which can save your quick notes.

### Running:
```
$ python menu.py
```

### An example:
```
        Notebook menu:
        1. Show notes
        2. Search notes
        3. Add a note
        4. Modify a note
        5. Remove a note
        6. Save notes localy
        7. Load local saved notes
        6. Quit
```
